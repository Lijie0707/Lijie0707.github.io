CentOS6系统,管理员可以使用如下指令来管理服务器的启动与停止

CentOS7-8系统,管理员可以使用systemctl命令来管理服务器启动与停止

systecmd目标名称
System V init运行级别 systemd目标名称 作用
0 runlevel0.target,poweroff.target 关机
1 runlevel1.target,rescue.target 单用户模式
2 runlevel2.target,multi-user.target 多用户的文本界面
3 runlevel3.target, multi-user.target
4 runlevel4.target, multi-user.target
5 runlevel5.target, graphical.target 多用户的图行界面
6 runlevel6.target, reboot.target 重启
6 7-8 作用
service NAME start systemctl start NAME 启动服务
service NAME stop systemctl stop NAME 停止服务
service NAME restart systemctl restart NAME 重启服务
service NAME reload systemctl reload NAME 重新加载配置（不终止服务）
service NAME status systemctl status NAME 查看服务状态
无 systemctl mask NAME 屏蔽服务运行
无 systemctl unmask NAME 取消屏蔽服务
6 7-8 作用
chkconfig NAME on systemctl enable NAME.service 开机自动启动
chkconfig NAME off systemctl disable NAME.service 开机不自动启动
chkconfig -list systemctl list-unit-files 查看各个级别下服务的启动与禁用
chkconfig -list NAME systemctl is-active NAME.service systemctl is-enabled 查看特定服务是否启动 查看特定服务是否为开机自启动
